The Future of Agentic AI: From Theory to Practice

Abstract

Recent advances in Large Language Models (LLMs) have paved the way for autonomous agents capable of complex reasoning and tool use. However, significant challenges remain in reliability, safety, and multi-agent collaboration.

In this seminar, we will discuss the scaling laws of agent capabilities and demonstrate our latest framework, AgentBeats, which achieves state-of-the-art performance on coding benchmarks.

T^2-Bench: Evaluating ConversationalAgents in a Dual-Control Environment

Abstract

Existing benchmarks for conversational AI agents simulate single-control environments, where only the AI agent can use tools to interact with the world, while the user remains a passive information provider. This differs from real-world scenarios such as technical support, where users need to actively participate in modifying the state of the (shared) world. To address this gap, we introduce T²-Bench, with four key contributions:

1) A novel telecom-inspired dual-control domain modeled as a Dec-POMDP, where both the agent and the user make use of tools to act in a shared, dynamic environment that tests both agent coordination and communication.

2) A compositional task generator that programmatically creates diverse, verifiable tasks from atomic components, ensuring domain coverage and controlled complexity.

3) A reliable user simulator tightly coupled with the environment, whose behavior is constrained by tools and observable states, improving simulation fidelity.

4) Fine-grained analysis of agent performance through multiple ablations, including separating errors arising from reasoning versus communication and coordination.

In particular, our experiments show significant performance drops when agents shift from no-user to dual-control settings, highlighting the challenges of guiding users. Overall, T²-Bench provides a controlled testbed for agents that must both reason effectively and guide user actions.

Test-Time Training

Abstract

We formulate long-context language modeling as a problem in continual learning, rather than architecture design. Under this formulation, we use only a standard Transformer architecture with sliding-window attention. However, the model continues learning at test time via next-token prediction on the given context, effectively compressing the context it reads into its weights.

In addition, we improve the model’s initialization for learning at test time through meta-learning during training. Overall, our method—Test-Time Training (TTT)—is end-to-end (E2E) at both test time (via next-token prediction) and training time (via meta-learning), in contrast to previous approaches.

We conduct extensive experiments with a focus on scaling properties. In particular, for 3B models trained with 164B tokens, TTT-E2E scales with context length in the same way as full-attention Transformers, while alternatives such as Mamba-2 and Gated DeltaNet do not. Meanwhile, similar to RNNs, TTT-E2E achieves constant inference latency regardless of context length, making it 2.7× faster than full attention for 128K context. Our code is publicly available.

Titans + MIRAS: Helping AI Have Long-term Memory

Abstract

Designing efficient and effective architectural backbones and optimization algorithms has long been central to improving the capabilities of foundation models. In this talk, we reconceptualize neural architectures and optimization algorithms as associative memory modules that learn key–value mappings using an internal objective, referred to as attentional bias. We observe that most existing sequence models rely on either dot-product similarity or L2 regression objectives as their attentional bias. Going beyond these objectives, we introduce a set of alternative attentional bias configurations and their effective approximations to stabilize training. We further reinterpret forgetting mechanisms in modern deep learning architectures as a form of retention regularization, and propose a novel set of forget gates for sequence models. Building on these insights, we present MIRAS, a general framework for designing deep learning architectures based on four design choices: (i) associative memory architecture, (ii) attentional bias objective, (iii) retention gate, and (iv) memory learning algorithm. We also introduce new sequence models, including Titans, which demonstrate exceptional performance on tasks such as language modeling, commonsense reasoning, and recall-intensive tasks—often outperforming Transformers and other modern linear recurrent models.
